Hexo 遇到的部署问题

hexo 遇到的部署问题

下面是一些常用的 hexo 命令

  • hexo g -d // 生成静态文件并部署
  • hexo s // 本地运行
  • hexo clean // 清理配置文件

出现问题的原因是 git 没有配置系统变量

本来系统是有的,因为win10更新导致系统变量好多 都丢失了 也不知道为什么 node 的环境变量也没了

直接上解决方案===>>>
解决方案一
重新配置环境变量 然后就可以了
解决方案二
到 hexo 的根目录下 右键 git bash ; 在git bash 里面执行 hexo d 就能完成部署了
